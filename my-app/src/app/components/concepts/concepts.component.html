code Decorator @NgModule() code No Angular é possível dividir a sua aplicação em módulos. Esses módulos podem ser contextos, fluxos, ou um grupo de componentes que possuem muita coisa em comum code Por padrão, uma aplicação tem pelo menos um módulo, que é o AppModule code Funcionalidades podem ser compartilhadas entre módulos code O framework oferece uma implementação de lazy-loading, onde você define quais módulos serão carregados no start da aplicação e quais devem ser carregados "on demand", melhorando bastante a performance code É altamente recomendável que uma aplicação complexa seja dividida em módulos code Decorator @Component() code Assim como os módulos, cada aplicação Angular tem pelo menos um componente, que é o AppComponent code O AppComponent é o "root component" da aplicação, ou seja, está no topo da hierarquia dos componentes code Template (HTML) code Style (CSS, SCSS, Sass, Less, Stylus) code Class (TypeScript) code Suporte à two-way-data biding code Decorator @Injectable() code Uma classe que pode ser compartilhada entre componentes code A dependência dos Components para os Services é implementada através de Injeção de Dependência code É altamente recomendável que toda a lógica de négocio, chamada à APIs, etc fiquem nos Services, uma vez que o seu Component ficaria responsável apenas pela interface, possibilitando facilmente a sua substituição code O framework oferece um módulo Router que é responsável pelo gerenciamento das rotas code Permite você definir rotas para a navegação entre componentes code O Router intercepta as solicitações de URLs e gerencia a exibição dos componentes code Se uma determinada rota precisa de um módulo que ainda não foi carregado, o Router pede o carregamento desse módulo através do lazy-loading code O Router também grava os logs de navegação no browser, portanto os botões de "Voltar" e "Avançar" continua funcionando normalmente, mesmo sem o recarregamento da página code É possível definir regras de navegação para cada rota, dando a possibilidade para proteção de rotas por exemplo
